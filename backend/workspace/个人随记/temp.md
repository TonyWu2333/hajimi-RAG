# 📝 笔记三：关于未来方向的一点思考

**时间：2026-03-27 01:10**

最近在想一个问题：
👉 我以后到底是做纯技术，还是偏产品？

从目前的情况来看，我有两个比较明显的特点：

### 1. 技术方面

* 会多种语言（Python / Java / C++）
* 能独立完成基本开发任务
* 对AI方向比较感兴趣

但问题是：

* 深度不够（很多只是“会用”）
* 对底层理解还比较弱

---

### 2. 产品方面

* 比较喜欢思考“用户为什么用这个功能”
* 做过一些需求分析和原型设计
* 也比较享受和别人讨论产品的过程

但问题是：

* 缺乏系统方法论
* 商业理解还比较浅

---

### 3. 当前的一个判断

我可能更适合走：
👉 **“技术 + 产品”的复合型路线（偏技术产品经理 or AI产品）**

原因：

* 纯开发竞争太激烈，而且需要很深的技术积累
* 纯产品又容易变成“空谈”
* AI方向本身就需要技术理解

---

### 4. 接下来计划

短期（3个月）：

* 把实习做好，争取转正
* 提升工程能力（代码质量、系统设计）

中期（1年）：

* 深入AI应用方向（RAG / Agent）
* 做1-2个完整项目（可展示）

长期：

* 能独立负责一个AI产品，从0到1

---

### 最后一点感受

现在最大的焦虑来源不是忙，而是“不确定”。
但反过来想，现在其实是试错成本最低的时候。

👉 先多尝试，再做选择，比一开始就选对更重要。
